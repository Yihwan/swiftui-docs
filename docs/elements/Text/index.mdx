## Decoration Modifers 
`underline()` and `strikethrough()` take optional `active` and `Color` params. 
<ContentBlock imgName="Text-decorationModifiers.png">
  <code className="clike">
    {`
/* 
    func bold() -> Text
    func italic() -> Text
    func underline(_ active: Bool = true, color: Color? = nil) -> Text
    func strikethrough(_ active: Bool = true, color: Color? = nil) -> Text
*/
VStack(alignment: .trailing, spacing: 10) {
    Text("Text Decoration Modifiers")
        .padding([.vertical])
    
    Text(".bold()")
        .bold()
    
    Text(".italic()")
        .italic()
    
    Text(".underline()")
        .underline()
    
    Text(".underline(color: .red)")
        .underline(color: .red)
    
    Text(".strikethrough()")
        .strikethrough()
    
    Text(".strikethrough(false)")
        .strikethrough(false)
}
.font(.title)
    `}
  </code>
</ContentBlock>

## foregroundColor()
Change text color 
```clike 
Text("Hello world").foregroundColor(.red) // makes text red
```

## Operations 
Check equality 
```clike 
"Hello" == "hello" // false 
"Hello" != "hello" // true 
```

Concatenate 
```clike
"Hello" + " " + "world" // "Hello world"
```

## Plain Text Label
```clike 
  Toggle("Toggle with just text", isOn: $areTogglesOn)
```

## Custom Label 
Create a custom label by passing a view as a callback. 
```clike 
Toggle(isOn: $areTogglesOn) {
    HStack {
        Text("Toggle with icon")
        Image(systemName: "pencil.circle.fill")
    }
}
```
